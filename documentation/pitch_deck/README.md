# Judge pitch deck

- `Sentinel_AI_Compact_Pitch_Deck.pptx` — concise seven-slide idea/market/next-steps pitch
- `Sentinel_AI_Compact_Pitch_Deck.pdf` — presentation-ready compact pitch
- `generate_compact_pitch_deck.py` — reproducible generator for the compact pitch
- `Sentinel_AI_Redesigned_Pitch_Deck.pptx` — redesigned 18-slide editable Sentinel AI deck
- `Sentinel_AI_Redesigned_Pitch_Deck.pdf` — redesigned presentation-ready PDF
- `redesigned_speaker_notes.md` — slide-by-slide delivery guide for the redesigned deck
- `generate_redesigned_deck.py` — reproducible LibreOffice/UNO generator for the redesigned deck
- `AI_Cyber_Threat_Intelligence_Pitch_Deck.pptx` — editable PowerPoint deck
- `AI_Cyber_Threat_Intelligence_Pitch_Deck.pdf` — presentation-ready PDF
- `speaker_notes.md` — 7–9 minute script, demo checklist, and judge Q&A
- `generate_pitch_deck.py` — reproducible LibreOffice/UNO generator

To regenerate the deck, start LibreOffice in headless listener mode on port 2002 and run the generator with the system Python that provides the `uno` module.
